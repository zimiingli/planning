# tab_method_classification

## Paper Location

Section 3.3 Main text. Design Implications (FLARE Table 5 style).

## Description

Classifies all methods by signal type, direction handling, Phase 1 requirement, extra inference cost, and SR win/loss record. DIAL is the only method with multi-signal + auto-learned direction + no Phase 1.

## Data Status

Complete.

## Data Source

Method properties from Section 1.2 of implementation plan; win/loss from Section 2.5.
