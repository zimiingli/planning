# fig_bsw_vs_rho

## Paper Location

Section 3.2 / Appendix, Proposition 1 Quantitative Validation.

## Description

Scatter plot with linear regression: |rho| of strongest signal vs BSW degradation (pp). Shows stronger signals lead to higher wrong-direction penalty.

## Data Status

Complete. r=0.90, R^2=0.80, p=0.104 (4 non-safe points). APPS Intv excluded from regression (rollout-safe). Note: p=0.104 due to only 4 data points -- acknowledged limitation.

Hardcoded from Section 3.3 / Section 3.8.4 in implementation plan (BSW SR data + signal discovery rho values).
