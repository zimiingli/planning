Core Insight

The same uncertainty signal reverses direction across environments.

Token entropy correlates negatively with rollout utility in FEVER (rho = -0.119) but positively in APPS Interview (rho = +0.317).

Two-Source Uncertainty Model

Type I (Info-Poverty): Agent lacks info; rollouts cannot help. High entropy = bad.

Type D (Decision-Diff): Agent has info but faces choices; rollouts explore alternatives. High entropy = good.

Instance of Simpson's Paradox in signal-utility space

Proposition (Necessity)

Direction discovery is a necessary condition for cross-environment non-negative value of computation. No fixed-direction gate can satisfy both Type I and Type D environments.
